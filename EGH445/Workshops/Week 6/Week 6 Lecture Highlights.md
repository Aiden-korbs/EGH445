## Core Idea
- The workshop is a recap-and-application session for **Discrete-time Control System Design 1**.
- It emphasises:
  - **system properties**
  - **design workflow**
  - **practical interpretation**
  - **applied state feedback design**

## High-Level Sequence
1. Check whether the system can be controlled and observed.
2. Build or obtain the discrete model.
3. Define performance targets.
4. Design the controller.
5. Evaluate the response.
6. Refine if needed.
