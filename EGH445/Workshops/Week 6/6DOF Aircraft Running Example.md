# 6DOF Aircraft Running Example

## Purpose
- The workshop uses a **6DOF aircraft model** as the main applied example.
- It connects:
  - modelling
  - assumptions
  - controllability/observability checks
  - controller design
  - response evaluation

## What, Why, How
| Question | Answer |
|---|---|
| **What?** | Real system, real system model, expected characteristics, direct controller design |
| **Why?** | Regulate behaviours, drive performance, simulate safely, implement controller |
| **How?** | Practical assumptions, mathematical techniques, evaluation |

## Important Note
- The workshop says the detailed application-specific equations are for **motivation and realism**.
- You are not expected to memorise all the aircraft equations.
- The main goal is understanding the **high-level control concepts**.
