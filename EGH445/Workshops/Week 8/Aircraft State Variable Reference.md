# Aircraft State Variable Reference

## Coordinate and Motion Variables
| Symbol | Meaning |
|---|---|
| $x, y, z$ | position coordinates |
| $u, v, w$ | velocity coordinates |
| $p$ | roll rate |
| $q$ | pitch rate |
| $r$ | yaw rate |
| $\phi$ | roll angle |
| $\theta$ | pitch angle |
| $\psi$ | yaw angle |
| $\beta$ | side-slip angle |
| $\alpha$ | angle of attack |

## Notes
- These variables are used throughout the 6DOF aircraft example.
- The workshop’s main worked example focuses on **longitudinal motion**, especially:
  - $u$
  - $w$
  - $\theta$
  - $q$
