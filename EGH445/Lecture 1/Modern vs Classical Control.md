## Comparison Table
| Feature            | Classical Control                                          | Modern Control                                                 |
| :----------------- | :--------------------------------------------------------- | :------------------------------------------------------------- |
| **Inputs/Outputs** |SISO (Single-Input Single-Output)   |MIMO (Multiple-Input Multiple-Output)   |
| **System Type**    |Linear, Time-Invariant (LTI)        |Nonlinear, Time-Varying                 |
| **Stability**      |Input-Output Stability              |Internal Stability                      |
| **Domain**         |Frequency Response                  |Time Response                           |

## Extended Capabilities of Modern Control
***State Feedback**: Design control with reduced state information.  
***Optimal Control**: Optimizing an objective function ($J$) subject to dynamics.  
***State Estimation**: Using **Observers** to infer internal states from output and reject disturbances.  
***Digital Control**: Design using computers involving A2D and D2A conversions.
